# Sales Performance Dashboard

## Project Overview
This project is a Tableau dashboard created using the Sample Superstore dataset. It analyzes sales and profit performance using different visualizations.

## Dashboard Includes
- Total Sales KPI
- Total Profit KPI
- Region Wise Sales Analysis
- Category Profit Analysis
- Sub-Category Profit Analysis

## Tools Used
- Tableau Desktop
- Sample Superstore Dataset

## Features
- Interactive dashboard
- Data visualization using bar charts
- Sales and profit insights
